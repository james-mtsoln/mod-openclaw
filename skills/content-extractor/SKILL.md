# Content Extraction & Repurposing

Turn one piece of long-form content into many platform-specific posts. Adapted from the [Content Extraction Skill](https://chemical-clematis-3fe.notion.site/content-extraction-skill-2efaade88814800382a1eedc015c0fff).

## Usage

```
/repurpose <article, transcript, or document>
```

## Instructions

### Step 1: Onboarding (first use)

Ask the user:

- **Target platforms**: Newsletter / Substack / Twitter/X / LinkedIn / YouTube Shorts
- **Preferred volume**: 3-5 ideas per platform? 5-10?
- **Brand voice**: professional / casual / provocative / educational

### Step 2: Deep Content Analysis

Analyze the source material and tag every section:

**High-Value Insights** 🔍

- Unique perspectives or contrarian takes
- "Aha moments" — things that surprised or challenged assumptions
- Novel connections between disparate ideas

**Teachable Content** 📚

- Frameworks and mental models
- Step-by-step processes
- Rules of thumb or heuristics

**Stories & Examples** 📖

- Personal experiences with emotional resonance
- Case studies with before/after
- Analogies that simplify complex ideas

**Data & Proof Points** 📊

- Statistics and numbers
- Comparisons and benchmarks
- Research citations

### Step 3: Multi-Platform Ideation

For each platform, generate ideas in the native format:

**Newsletter** → Long-form thought leadership

- Opening hook + deep insight + practical takeaway

**Twitter/X** → Thread or standalone tweet

- Hot take + supporting evidence + call for discussion
- Thread: 1/ hook → 2-5/ points → 6/ summary + CTA

**LinkedIn** → Professional insight post

- "I used to think X. Then I learned Y." format
- Data-backed industry observations

**YouTube Shorts** → 30-60 second script

- Hook (5s) → Core insight (20-40s) → CTA (5s)

### Step 4: Present for Selection

```
## Platform: {name}

### Idea 1: {headline}
- **Source section:** {which part of the original}
- **Format:** {thread / single / article}
- **Draft:**
{actual draft content}
```

### Rules

- **Never copy-paste** — always transform for the platform's native style
- **One source insight can become multiple platform posts** — that's the point
- **Always credit the source** when repurposing others' content
- **Prioritize "Aha moments"** — unique insights outperform generic advice
